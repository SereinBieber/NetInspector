# NetInspector
## 基于 Python 的多线程端口扫描与安全评估系统

> 四川大学网络空间安全学院《计算机通信与网络》课程设计项目

---

## 项目简介

NetInspector 是一个基于 Python 开发的轻量级多线程端口扫描与安全评估系统，旨在实现网络主机发现、端口扫描、服务识别及基础安全检测等功能。

本项目作为四川大学网络空间安全学院《计算机通信与网络》课程设计作品完成，在满足课程基本要求的基础上，实现了服务探测、风险提示以及扫描报告导出等扩展功能，构建了一个具有实际应用价值的小型网络扫描工具。

指导教师：王海舟老师

---

## 项目目标

本项目参考 Nmap 等经典网络扫描工具的设计思想，实现以下功能：

### 基础功能

- [x] 判断目标主机是否存活；
- [x] 支持扫描目标主机的开放端口；
- [x] 支持设置目标 IP 地址范围；
- [x] 支持设置端口范围或指定多个端口；
- [x] 支持自定义扫描超时时间；
- [x] 支持自定义扫描线程数量；
- [x] 支持扫描结果实时显示。

### 扩展功能

- [x] 支持导出 TXT 扫描报告；
- [x] 支持导出 HTML 扫描报告；
- [x] 支持常见服务识别与版本探测；
- [x] 支持基础安全风险检测；
- [x] 支持简单操作系统信息推测（可选）。

## 使用说明

1. 输入目标 IP 地址或 IP 范围；
2. 设置扫描端口范围；
3. 设置线程数量与超时时间；
4. 点击“开始扫描”；
5. 查看扫描结果及风险提示；
6. 根据需要导出 TXT 或 HTML 扫描报告。
